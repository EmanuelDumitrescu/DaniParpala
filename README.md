# My Site - Getting Started Guide

## Project Structure
```
ProiectSite/
├── index.html          (Main page)
├── css/
│   └── styles.css      (Styling)
├── js/
│   └── script.js       (JavaScript functionality)
├── images/             (Place your images here)
└── README.md           (This file)
```

## How to Use

1. **Open the site**: Double-click `index.html` to open it in your default browser, or right-click and select "Open with" to choose a browser.

2. **Edit content**: 
   - Open `index.html` in VS Code
   - Change the text, sections, and links to match your content
   - Save the file (Ctrl+S)
   - Refresh the browser (F5) to see your changes

3. **Customize colors and styles**:
   - Open `css/styles.css` in VS Code
   - Modify colors, fonts, spacing, and layouts
   - The changes will appear when you refresh the browser

4. **Add interactivity**:
   - Open `js/script.js` in VS Code
   - Add more functions and event listeners
   - Refresh the browser to test

## Common Customizations

### Change the site title
In `index.html`, find this line and change "My Site" and "Welcome to My Site":
```html
<title>Welcome to My Site</title>
```

### Change the main color
In `css/styles.css`, search for `#007bff` (blue) and replace with your color code.
You can find color codes at [colorpicker.com](https://www.colorpicker.com)

### Add a new section
Copy an existing section in `index.html`, paste it below, and customize the content.

### Add images
1. Place image files in the `images/` folder
2. In `index.html`, use: `<img src="images/filename.jpg" alt="Description">`

## Tips for Beginners

- **Test frequently**: Keep the browser open and refresh after each change
- **Use browser DevTools**: Right-click → Inspect to see what's happening (useful for debugging)
- **Start simple**: Don't try to build everything at once
- **Mobile-first thinking**: Check how it looks on your phone by resizing the browser window
- **Keep backups**: Before major changes, save a copy of your files

## Deploy Your Site

When you're ready to go live:
1. Upload all files to a web host (like GitHub Pages, Netlify, or traditional hosting)
2. Make sure the folder structure stays the same
3. Upload `index.html` as the main file
